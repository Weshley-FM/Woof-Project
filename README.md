# Woof - AI & Code Generator Web Template

Welcome to the **Woof** source code repository!  
This is a premium, modern, and responsive website template designed specifically for AI startups, coding platforms, and SaaS products.

## 📂 Repository Structure

This project has been packed in such a way that it is easy to handover, learn, and customize:

- `Woof/` - Core source code of the web application built with React JS, Vite, and Tailwind CSS.
- `03_Design_Asset/` - Links and blueprint assets for the user interface design (Figma).
- `Screenshots/` - Preview images of the final web application page layouts.

## 🚀 Quick Start

If you want to try running the application directly on your local machine:

1. Make sure you have **Node.js** installed (version 18 or higher).
2. Open your terminal and navigate to the source code folder: 
   ```bash
   cd "Woof - AI&Code Generator"
   ```
3. Install the project packages / dependencies:
   ```bash
   npm install
   ```
4. Run the local development server:
   ```bash
   npm run dev
   ```
   Open your browser and access the local link shown in the terminal (typically http://localhost:5173).
